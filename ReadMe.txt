Instruction on how to run the code successfuly in 3 easy steps

(All the operations must be performed in MATLAB R2019b version or higher!)

Step 1. Extract all the files form the .zip file to selected location
Step 2. Open FigDataExtract.m using MATLAB R2019b, run it and wait for files to extract.
Step 3. Open gui.m file and run it

Now you can use the slider to navigate and switch between the different hours.
